# Section 3: Git Integration (Lessons 13-17)

## Lesson 13: Git Basics with Claude

**Objectives**: Check status, view diffs, understand staging, review commits

**Key Commands**:
```
"Show git status"
"Show me the diff"
"What commits were made recently?"
```

**Exercise**: Navigate a git repository with Claude's help

---

## Lesson 14: Making Commits

**Objectives**: Create commits, write good messages, use Co-Authored-By

**Workflow**:
```
1. "Show git status"
2. "Stage these files: [list]"
3. "Create a commit with message: [description]"
```

**Exercise**: Make your first commit with Claude

---

## Lesson 15: Branch Management

**Objectives**: Create, switch, merge, delete branches

**Commands**:
```
"Create branch feature-x"
"Switch to branch feature-x"
"Merge feature-x into main"
```

**Exercise**: Complete feature branch workflow
