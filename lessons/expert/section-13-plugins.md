# Section 13: Plugins (Lessons 81-86)

## Lesson 81: Plugin Architecture

**Components**:
- Skills directory
- Subagents directory
- Hooks configuration
- MCP servers
- README and metadata

**Structure**:
```
my-plugin/
├── skills/
├── subagents/
├── hooks/
├── mcp/
├── README.md
└── plugin.json
```

## Lesson 82: Creating Plugins

**Steps**:
1. Design plugin scope and features
2. Create directory structure
3. Build skills, hooks, subagents
4. Add MCP servers if needed
5. Document thoroughly
6. Test on real projects

**Exercise**: Build your first plugin

## Lesson 83: Plugin Distribution

**Publishing**:
1. Package plugin files

4. Publish to registry or GitHub
